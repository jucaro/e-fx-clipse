From creation to deployment Your first JavaFX app

The following tutorial will guide you from the creation of a JavaFX project to its final export and deployment

1.

File > New > Project ...

2.

Select the "JavaFX Project" entry

3.

Enter a project name e.g. "MyJavaFXApp" and click "Finish"

4.

Create a package e.g. "myapp"

5.

Open the context menu on the "myapp" package and select New > Other ...

6.

Select "JavaFX Main Class"

7.

Enter a name e.g. "MyApplication"

8.

Add some JavaFX code

9.

Set the selection on the "MyApplication" entry and hit the "Run"-Toolbar item

10.

Open the build.fxbuild file and fill in the following fields:

  • Vendor name
  • Application version
and select the "Browse..." button next to the "Application class" entry

11.

Click on the "ant build.xml and run" Link in the Build section of the editor

12.

Expand the folders in the "Package Explorer" and double click on "MyJavaFXApp.jar"
